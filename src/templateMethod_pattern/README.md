##머리감기를 단계별로 적어보자
  1. 샤워기를 튼다.(샤워기밖에없다는 가정!)
  2. 머리에 물을 적신다.
  3. 머리에 샴푸나 비누등으로 거품을 내준다.
  4. 마사지를 해준다.
  5. 샤워기로 거품을 깨끗하게 행궈준다.
