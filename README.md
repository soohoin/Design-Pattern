    - 템플릿 메서드와 전략 패턴의 조합  
 3. 상태(State) 패턴
 4. 데코레이터(Decorator) 패턴

    - 옵저버 패턴 구현의 고려 사항
 8. 미디에이터(Mediator) 패턴  
 9. 파사드(Facade) 패턴
10. 추상 팩토리(Abstract Factory) 패턴
11. 컴포지트(Composite) 패턴
